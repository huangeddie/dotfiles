# Note Types

## Daily Notes

**Location:** `Daily/YYYY-MM-DD.md`

**Rules:**
- Empty by default
- Exist solely as linking targets from other notes
- Never write prose or journal entries directly in daily notes
- Created automatically or on demand when needed as a link target

## Journal Entries

**Location:** Root folder

**Characteristics:**
- Stream of consciousness cataloging recent events
- Profuse internal links to people, places, books, ideas mentioned
- Often a "day in review" format
- Can be dated in filename or linked to daily note

## Evergreen Notes

**Location:** Root folder

**Characteristics:**
- Atomic, single-idea notes
- Permanently useful, not tied to a specific time
- Heavily linked to other evergreen and reference notes
- Written in my own words, not clipped from elsewhere

## Reference Notes

**Location:** `References/`

**Subtypes:**
- **Books** — `Book Title.md`
- **Movies** — `Movie Title.md`
- **People** — `Person Name.md`
- **Places** — `Place Name.md`
- **Podcasts** — `Podcast Name.md` or `Episode Title.md`

Each uses a standard template with consistent frontmatter.

## Clippings

**Location:** `Clippings/`

**Characteristics:**
- Content written by others
- Saved from web via Obsidian Web Clipper or manual paste
- Usually preserve original source URL
- My commentary (if any) goes in a separate section or separate note

## Archives

**Location:** `Archives/`

**Characteristics:**
- Legacy imports from previous note-taking systems
- Treat as read-only when possible
- Migrate active, relevant notes to proper locations (Root, References, Clippings)
- Do not create new notes in Archives
